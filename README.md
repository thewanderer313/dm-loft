# DM Loft

Web hub for D&D DM tools. Vercel + Supabase + Next.js 15.

## Local dev
- `npm run dev` — start dev server on :3000
- `npm test` — run unit tests

## Deploys
- Push to `main` → Vercel deploys production
- Push to any other branch → Vercel deploys a preview
